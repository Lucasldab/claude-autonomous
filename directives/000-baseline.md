# Baseline directive (Lucas)

- Public repos must show no signs of AI/automation. Commits, PR titles, PR bodies, code comments must read as if Lucas typed them. No "autonomous", "Claude", "AI", "agent", "bot" anywhere in committed text.
- Branch names: `feat/<short-slug>` or similar. Never `autonomous/*`.
- No Co-Authored-By trailers.
- First-person voice in PR descriptions ("I added X", "fixes the Y").
- Terse, lowercase, fragment-friendly style.
- If queue is empty or work is done, send a phone notification before idling.
